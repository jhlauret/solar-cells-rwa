# smart-contracts

Emplacement réservé aux contrats intelligents.

## Statut MVP
**Les wallets sont custodial au MVP.** Aucun smart contract n'est déployé ou déployable en l'état.

Ce dossier existe pour préparer la phase post-MVP : représentation on-chain des actifs solaires, transferts whitelistés au niveau du contrat, distribution de revenus.

## Outillage envisagé
- Hardhat **ou** Foundry (à trancher dans une spec dédiée).
- Tests on-chain isolés.
